# coflow控制流语法设计

coflow控制流采用现代C-like块语法
使用花括号表示代码块
不使用lua的then，do，end
不使用python的缩进语义

# 设计原则
1. 语法现代
2. 容易解析
3. 容易实现
4. 少特殊规则
5. 适合嵌入式脚本语言

# if语句
if用于条件分支

```coflow
if hp <= 0 {
  die()
} else {
  update()
}
```

支持else if

```coflow
if score >= 90 {
  rank = "S"
} else if score >= 60 {
  rank = "A"
} else {
  rank = "B"
}
```

# while循环
while用于条件循环

```coflow
while running {
  update()
}
```

# for循环
for使用for-in作为统一循环语法

```coflow
for item in items {
  print(item)
}
```

数值循环使用区间表达式

```coflow
for i in 1..10 {
  print(i)
}
```

字典遍历支持key，value

```coflow
for key, value in scores {
  print(key, value)
}
```

复杂范围使用range函数

```coflow
for i in range(1, 10, 2) {
  print(i)
}
```

# break和continue
break跳出当前循环
continue进入下一次循环

```coflow
for enemy in enemies {
  if enemy.dead {
    continue
  }

  if enemy.boss {
    break
  }
}
```

# return语句
return从函数返回

```coflow
fn add(a: int, b: int) -> int {
  return a + b
}
```
