# coflow函数语法设计

函数是一等值
支持普通函数和匿名函数
参数类型和返回类型可选
第一版暂不支持闭包捕获

# 普通函数
使用fn声明普通函数

```coflow
fn add(a: int, b: int) -> int {
  return a + b
}
```

参数类型可选

```coflow
fn print_value(value) {
  print(value)
}
```

返回类型可选

```coflow
fn add(a, b) {
  return a + b
}
```

# 匿名函数
匿名函数可以作为值使用

```coflow
var add = fn(a: int, b: int) -> int {
  return a + b
}
```

函数值可以作为配置常量值

```coflow
skill = {
  id: "fireball"
  apply: fn(caster: Entity, target: Entity) -> void {
    target.hp -= 10
  }
}
```

# 函数类型
函数类型使用fn签名表示

```coflow
var damage_fn: fn(attacker: Entity, target: Entity) -> int
```

class字段可以使用函数类型

```coflow
class Skill {
  id: string
  apply: fn(caster: Entity, target: Entity) -> void
}
```

# 命名参数
支持命名参数调用

```coflow
spawn(id: "slime", x: 10, y: 5)
```

普通位置参数仍然可用

```coflow
spawn("slime", 10, 5)
```

# 返回值
return从函数返回

```coflow
fn get_hp(player: Entity) -> int {
  return player.hp
}
```

第一版不支持lua式多返回值

不支持：

```coflow
return x, y
```

# 闭包
第一版暂不支持闭包捕获

允许：

```coflow
var add_one = fn(x: int) -> int {
  return x + 1
}
```

暂不支持：

```coflow
var base = 10

var add_base = fn(x: int) -> int {
  return x + base
}
```

# 暂不支持
1. 闭包捕获
2. lua式多返回值
3. 函数重载
4. 默认参数
5. 可变参数
