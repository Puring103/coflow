# coflow提案目录

本目录保存尚未进入核心语言的语法和语义提案。

核心规则：

1. 核心文档只描述已经决定的最小语言。
2. 未定能力不写进核心规格。
3. 每个提案独立说明动机，语法，语义冲突和实现成本。
4. 提案成熟后再合并进核心文档。
