# coflow表达式与运算符设计

coflow表达式语法应清晰，脚本化，容易解析
逻辑运算使用单词形式
不使用C风格的`&&`，`||`，`!`

# 字面量
支持基础字面量

```coflow
123
3.14
true
false
null
"hello"
```

数字字面量支持`_`作为分隔符
分隔符只提升可读性，不影响数值

```coflow
1_000
1_000_000
3.141_592
```

`_`不能出现在数字开头，结尾或连续出现

# 容器字面量
支持数组，字典，对象字面量

```coflow
var nums = [1, 2, 3]

var scores = {
  "alice": 10,
  "bob": 20,
}

weapon = {
  id: "sword",
  damage: 10,
}
```

对象字面量支持字段简写
`id`等价于`id: id`

```coflow
var id = "sword"
var damage = 10

weapon = {
  id,
  damage,
}
```

字段简写只允许简单标识符
不支持`player.name`或`items[0]`形式的字段简写

# 访问表达式
支持字段访问，索引访问，可选链

```coflow
player.hp
items[0]
scores["alice"]
player?.weapon?.name
```

支持可选索引
左侧为null或缺失时结果为null

```coflow
scores?["alice"]
items?[0]

var score = scores?["alice"] ?? 0
```

# 切片表达式
索引访问支持切片
切片使用范围表达式语义

```coflow
items[1..4]
items[..3]
items[3..]
items[..]
name[0..1]
```

`a..b`切片默认左闭右开
`a..=b`切片为左闭右闭
省略起点表示从0开始
省略终点表示直到末尾

```coflow
items[0..items.length]
items[0..=2]
```

# 对象展开
对象字面量支持展开
后面的字段覆盖前面的字段

```coflow
strong_sword = {
  ...base_sword,
  damage: 20,
}
```

# 数组展开
数组字面量支持展开
展开项按顺序插入数组元素

```coflow
all_items = [
  ...base_items,
  sword,
  shield,
]
```

数组展开要求右侧是可迭代值
配置常量表达式中，数组展开的右侧必须是编译期可确认的数组或常量可迭代值

# 调用表达式
支持位置参数调用

```coflow
spawn("slime", 10, 5)
```

支持命名参数调用

```coflow
spawn(id: "slime", x: 10, y: 5)
```

# 算术运算
支持常见算术运算

```coflow
a + b
a - b
a * b
a / b
a % b
```

# 字符串拼接
字符串拼接使用`+`

```coflow
var text = "hello " + name
```

# 比较运算
支持比较运算

```coflow
a == b
a != b
a < b
a <= b
a > b
a >= b
```

# in成员判断
in用于判断成员关系

```coflow
if item in inventory {
  use(item)
}

if "hp" in stats {
  print(stats["hp"])
}
```

not in用于反向判断

```coflow
if item not in inventory {
  add(item)
}
```

# 逻辑运算
逻辑运算使用and，or，not

```coflow
if alive and hp > 0 {
  update()
}

if dead or hp <= 0 {
  respawn()
}

if not visible {
  show()
}
```

# is类型判断与收窄
is用于类型判断
is可以在if分支中进行类型收窄

```coflow
fn print_id(id: int | string) {
  if id is string {
    print(id.length)
  } else {
    print(id + 1)
  }
}
```

第一版只支持简单形式的类型收窄

```coflow
if value is Type {
  // value narrowed to Type
}
```

支持is not

```coflow
if value is not string {
  return "value must be string"
}
```

暂不支持复杂布尔表达式中的高级收窄

# 范围表达式
范围表达式默认左闭右开

```coflow
0..10
```

表示包含0，不包含10

闭区间使用`..=`

```coflow
1..=10
```

表示包含1，也包含10

范围表达式可以用于for循环

```coflow
for i in 0..items.length {
  print(items[i])
}
```

范围表达式也可以用于切片

```coflow
var visible_items = items[0..10]
```

# if表达式
if可以产生表达式值

```coflow
var state = if hp <= 0 {
  "dead"
} else {
  "alive"
}
```

if表达式必须包含else分支
详细控制流规则见《2-控制流语法.md》

# match表达式
match可以按值或类型选择表达式值

```coflow
var text = match state {
  "idle" => "waiting",
  "run" => "moving",
  _ => "unknown",
}

var kind = match value {
  is int => "number",
  is string => "text",
  _ => "unknown",
}
```

详细控制流规则见《2-控制流语法.md》

# 空值合并
空值合并使用`??`

```coflow
var name = player.name ?? "unknown"
var damage = weapon.damage ?? 0
```

只有左侧为null或缺失时才使用右侧

# 赋值运算
支持普通赋值和复合赋值

```coflow
a = b
hp += 10
hp -= 5
count *= 2
speed /= 2
count %= 2
name ??= "unknown"
```

`??=`表示左侧为null或缺失时才赋值

# 运算符优先级
从高到低：

1. 调用，字段访问，索引访问，可选链，可选索引
2. 切片
3. 对象展开，数组展开
4. 一元`not`，一元`-`
5. `*`，`/`，`%`
6. `+`，`-`
7. 范围`..`，`..=`
8. `<`，`<=`，`>`，`>=`
9. `in`，`not in`
10. `is`，`is not`
11. `==`，`!=`
12. `and`
13. `or`
14. `??`
15. if表达式，match表达式
16. 赋值，复合赋值，`??=`

# 暂不支持
1. `&&`
2. `||`
3. `!`
4. 三元表达式
5. 位运算
6. 管道符`|>`
7. 复杂模式匹配
8. match守卫条件
