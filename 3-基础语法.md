# coflow基础语法设计

coflow基础语法以现代脚本语言为主
语法应简单，清晰，容易解析

# 注释
支持单行注释

```coflow
// this is a comment
var hp = 100
```

支持块注释

```coflow
/*
this is a block comment
*/
var hp = 100
```

暂不支持嵌套块注释

# 变量声明
局部变量使用var

```coflow
var hp = 100
var name: string = "hero"
var target: Entity? = null
```

var为块级作用域

```coflow
if alive {
  var hp = 100
}
```

顶层var为文件内辅助变量
不自动视为配置

```coflow
var version = "1.0"
```

# 配置定义
顶层`name = value`定义自动视为配置
顶层配置可以选择添加类型标注
配置是顶层常量定义，不是变量

```coflow
sword = {
  id: "sword"
  damage: 10
}

sword: Weapon = {
  id: "sword"
  damage: 10
}
```

# 顶层配置
不带var的顶层定义是配置
配置不是普通变量

```coflow
hp = 100
hp: int = 100
```

配置项默认对宿主可见
建议工具链对疑似拼写错误的顶层配置给出诊断

# 普通赋值
支持普通赋值

```coflow
var hp = 100
hp = hp - 10
```

# 字段赋值
支持字段赋值

```coflow
player.hp = 50
```

# 索引赋值
支持索引赋值

```coflow
items[0] = "sword"
scores["alice"] = 100
```

# 复合赋值
支持常见复合赋值

```coflow
hp += 10
hp -= 5
count *= 2
speed /= 2
```

# 不支持多重赋值
第一版不支持lua式多重赋值

不支持：

```coflow
a, b = b, a
```

# 不支持自增自减
第一版不支持++和--

不支持：

```coflow
i++
i--
```

使用复合赋值：

```coflow
i += 1
i -= 1
```

# 可选链
支持可选链访问

```coflow
var name = player?.weapon?.name
```

如果中间值为null或缺失，结果为null

# 空值合并
支持空值合并

```coflow
var name = player.name ?? "unknown"
var damage = weapon.damage ?? 0
```

只有左侧为null或缺失时才使用右侧

# 尾随逗号
数组，字典，对象字面量和参数列表允许尾随逗号

```coflow
var nums = [
  1,
  2,
  3,
]

sword = {
  id: "sword",
  damage: 10,
}

spawn(
  id: "slime",
  x: 10,
  y: 5,
)
```

# 字符串插值
支持字符串插值

```coflow
var text = "hello ${name}"
```
