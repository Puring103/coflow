# coflow基础语法设计

coflow基础语法以现代脚本语言为主
语法应简单，清晰，容易解析

# 注释
支持单行注释

```coflow
// this is a comment
var hp = 100
```

支持块注释

```coflow
/*
this is a block comment
*/
var hp = 100
```

暂不支持嵌套块注释

# 变量声明
局部变量使用var

```coflow
var hp = 100
var name: string = "hero"
var target: Entity | null = null
```

var为块级作用域

```coflow
if alive {
  var hp = 100
}
```

顶层var为普通顶层变量
默认公开
不自动视为配置

```coflow
var version = "1.0"
local var cache_version = "1.0"
```

# 配置定义
顶层`name = value`定义自动视为配置
顶层配置可以选择添加类型标注
配置是顶层常量定义，不是变量

```coflow
sword = {
  id: "sword"
  damage: 10
}

sword: Weapon = {
  id: "sword"
  damage: 10
}
```

# 顶层配置
不带var的顶层定义是配置
配置不是普通变量

```coflow
hp = 100
hp: int = 100
```

配置项默认对宿主可见
建议工具链对疑似拼写错误的顶层配置给出诊断

# 普通赋值
支持普通赋值

```coflow
var hp = 100
hp = hp - 10
```

# 字段赋值
支持字段赋值

```coflow
player.hp = 50
```

# 索引赋值
支持索引赋值

```coflow
items[0] = "sword"
scores["alice"] = 100
```

# 复合赋值
支持常见复合赋值

```coflow
hp += 10
hp -= 5
count *= 2
speed /= 2
```

# 解构绑定
var声明支持数组解构和对象解构
解构只用于绑定新变量，不用于赋值已有变量

```coflow
var [x, y] = position
var [first, second, ...rest] = items

var { id, damage } = weapon
var { name, cooldown = 1.0 } = weapon
```

对象解构按字段名取值
字段不存在且没有默认值时，结果为缺失值诊断或运行时错误
带类型上下文的解构按目标字段类型检查

```coflow
var { x: px, y: py } = player.position
```

数组解构按索引取值
`...rest`只能出现在数组解构的最后一项

```coflow
var [head, ...tail] = items
```

for循环支持解构绑定

```coflow
for { id, damage } in weapons {
  print(id, damage)
}

for [x, y] in points {
  draw_point(x, y)
}
```

第一版不支持嵌套解构
第一版不支持解构赋值

# 不支持多重赋值
第一版不支持lua式多重赋值

不支持：

```coflow
a, b = b, a
```

# 不支持自增自减
第一版不支持++和--

不支持：

```coflow
i++
i--
```

使用复合赋值：

```coflow
i += 1
i -= 1
```

# 可选链
支持可选链访问

```coflow
var name = player?.weapon?.name
```

如果中间值为null或缺失，结果为null

# 空值合并
支持空值合并

```coflow
var name = player.name ?? "unknown"
var damage = weapon.damage ?? 0
```

只有左侧为null或缺失时才使用右侧

# 尾随逗号
数组，字典，对象字面量和参数列表允许尾随逗号

```coflow
var nums = [
  1,
  2,
  3,
]

sword = {
  id: "sword",
  damage: 10,
}

spawn(
  id: "slime",
  x: 10,
  y: 5,
)
```

# 字符串插值
支持字符串插值

```coflow
var text = "hello ${name}"
```

# 原始字符串
原始字符串不处理转义字符
适合路径，正则，shader片段和宿主DSL

```coflow
var path = r"C:\game\assets\hero.png"
var pattern = r"\d+\.\d+"
```

# 多行字符串
支持三引号多行字符串
多行字符串保留换行

```coflow
var text = """
line one
line two
"""
```

原始多行字符串使用`r"""`开始

```coflow
var shader = r"""
void main() {
  // shader source
}
"""
```
