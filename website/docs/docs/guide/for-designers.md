# 给策划的配置维护路径

本页面向配置策划、系统策划和内容维护者，说明 Coflow 如何支持日常配置生产。

## 需要回答的问题

- 哪些数据适合继续放在 Excel 或飞书表格。
- 什么时候使用 CFD 文本配置维护复杂结构。
- record key、字段、引用和默认值如何理解。
- 出错时如何根据诊断定位到文件、sheet、行列、record 和字段。
- 可视化编辑器、VS Code/LSP 和 AI agent 分别能辅助哪些工作。
