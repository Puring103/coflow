# 提案：对象与字典扩展

核心版本区分对象和字典：

1. 对象是静态字段集合，字段异构，点访问。
2. 字典是动态键值集合，值同构，索引访问。

本提案讨论额外构造能力。

## 动态key字面量

```coflow
var key = "alice"

scores = {
  [key]: 10,
}
```

## 对象展开

```coflow
strong_sword = {
  ...base_sword,
  damage: 20,
}
```

## 数组展开

```coflow
items = [
  ...base_items,
  "sword",
]
```
