# coflow协程语法设计

coflow支持轻量协作式协程
协程用于游戏脚本，工具脚本，流程控制和宿主调度
第一版采用stackful coroutine

# 设计原则
1. 语法显式
2. 协作式调度
3. 不做抢占式线程
4. 宿主负责调度器
5. 语言只提供协程对象，yield，resume
6. co fn显式标记协程入口

# 协程函数
使用co fn声明协程函数
co fn是协程入口函数

```coflow
co fn blink(entity: Entity) {
  while true {
    entity.visible = false
    yield
    entity.visible = true
    yield
  }
}
```

调用co fn不会立即执行完
调用结果是coroutine对象

```coflow
var routine = blink(player)
routine.resume()
routine.resume()
```

# 匿名协程函数
支持匿名co fn

```coflow
var task = co fn() {
  yield
}
```

协程函数可以作为配置常量值

```coflow
cutscene = {
  run: co fn() {
    yield wait.seconds(1.0)
  }
}
```

# yield语句
yield只能在协程上下文中执行
协程上下文由co fn创建
yield可以出现在co fn直接或间接调用的脚本函数中
yield是语句，不是表达式

```coflow
co fn task() {
  yield
  yield 1
  yield "wait"
}
```

无值yield等价于yield null

# resume结果
resume返回结果对象

```coflow
{
  done: bool
  value: any
}
```

示例：

```coflow
co fn counter() {
  yield 1
  yield 2
}

var c = counter()

c.resume() // { done: false, value: 1 }
c.resume() // { done: false, value: 2 }
c.resume() // { done: true, value: null }
```

# return语句
co fn中的return表示协程最终结果

```coflow
co fn task() -> int {
  yield 1
  return 10
}
```

最后一次resume返回：

```coflow
{ done: true, value: 10 }
```

# yield协程自动委托
yield可以接收另一个coroutine对象
当yield的值是coroutine时，当前协程自动委托执行子协程
子协程完成后，父协程继续执行

```coflow
co fn child() {
  yield 1
  yield 2
}

co fn parent() {
  yield child()
  yield 3
}
```

执行结果：

```coflow
var c = parent()

c.resume() // { done: false, value: 1 }
c.resume() // { done: false, value: 2 }
c.resume() // { done: false, value: 3 }
c.resume() // { done: true, value: null }
```

# 普通函数边界
co fn可以调用普通函数
普通函数在协程上下文中可以yield
yield会暂停整个协程调用栈

```coflow
fn log_start() {
  print("start")
}

fn wait_click() {
  yield wait.event("click")
}

co fn task() {
  log_start()
  wait_click()
}
```

普通函数在非协程上下文中执行yield是错误

```coflow
fn helper() {
  yield // 只能在协程上下文中执行
}

helper() // 错误
```

# 已完成协程
已完成协程再次resume返回done结果

```coflow
{ done: true, value: null }
```

# 错误处理
协程内部发生错误时
resume传播运行时错误
协程进入dead状态

```coflow
co fn task() {
  throw "failed"
}

var c = task()

try {
  c.resume()
} catch err {
  print(err.message)
}
```

# 宿主调度
语言核心不内置等待秒数，等待帧，等待事件
这些能力由宿主或标准库提供

```coflow
co fn cutscene() {
  yield wait.seconds(1.0)
  yield wait.event("click")
  yield wait.frame()
}
```

# 暂不支持
1. 抢占式线程
2. async/await
3. yield作为表达式
4. resume传值回yield
